02_WEB

GENEREL

Med de første spæde skridt ind i web-udviklings verdenen, fokuserede vi to-delt på hhv. HTML & CSS, samt design af elementer.
I designdelen handlede det om designteori inden for fonte (Serif, Sans-serif, slab-serif etc), billedekomposition og farvesammensætning. Det udmuntrede i design af logo, favicon og splashbillede til første website.
I web-udviklings delen startede vi med at blive bekendt med grundlæggende HTML og CSS, hvor vi opsatte vores første hjemmeside herunder fagtermer (header, nav, splashbilleder etc.), som er grundlæggende inden for fagområdet. Grundlæggende var en forståelse for html skeletetet og dernæst placering af elementer med GRID & FLEX.



KEY ELEMENTS

BASIC HTML - BASIC CSS - POSSITIONING (Grid & Flex) - TILFØJELSE AF FONTE - LOGO & FAVICON - DESIGNTEORI TIL BILLEDER - WIREFRAME - LAYOUTDIAGRAM - TYPOGRAFI - DEBUGGING - SEMATIK - UPLOAD TIL SERVER



Valg af metode

I den afsluttende opgave, lavede jeg et website ud fra udleveret wireframe og layourdiagram, hvor jeg særligt kunne sætte mit præg ved at fokusere på design af splashbillede, logo og farvekombinationen på sitet.
